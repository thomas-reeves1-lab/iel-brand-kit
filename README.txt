Typeface
  Arial / Helvetica (bold for headings, regular for body)

Folders
  Icons/      47 custom icons (256x256 PNG, transparent rounded corners)
  Brand/      brand board, colour palette, seamless pattern, circular
              avatar, A4 letterhead, email signature
  Banners/    Notion cover header, slim PDF strip, square app icon
  Previews/   icon contact sheets (3 batches)
  Logo/       primary logo (white on navy)

How to use
  Notion  - upload an icon as a page icon (Add icon > Upload), or the
            header as the page cover.
  PDFs    - drop the slim strip at the top of policy documents.
  Email   - use the signature card; replace the placeholder details.
  Social  - use the circular avatar for profile pictures.

All assets re-generate from scripts in the IEL invoicing engine repo
(scripts/build_*.py) - change text, colours or sizes and re-render.
